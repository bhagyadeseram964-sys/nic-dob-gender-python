# NIC DOB & Gender Extractor 🇱🇰

A simple Python script to extract Date of Birth and Gender
from **Sri Lankan NIC numbers** (Old & New formats).

---

## Features
- Supports **Old NIC** (10 digits + V/X)
- Supports **New NIC** (12 digits)
- Automatically detects **Gender**
- Calculates **Date of Birth**
- Handles **Leap Years**

---

## Technologies Used
- Python
- datetime module

---

## How to Run
1. Make sure Python is installed on your system
2. Clone this repository or download the ZIP
3. Open a terminal in the project folder
